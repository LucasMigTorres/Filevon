# Fileon
Fileon is a simple shortcut manager intended to make access to your favorite files and apps convenient.
Add the directories of your frequently used items to the app for ease of access.
Folders, files, and apps are separated into three lists and can be manually sorted within each.

# Installation
Currently, Fileon is only available for Windows 11. Support for other operating systems may be implemented in the future.

There is no setup process, as the only download required for the app to work is the exe.

Because this application runs as an executable, Windows is likely to flag it as suspicious upon download or when running for the first time despite it not being malicious.

# Usage
To add folder, file, or app shortcuts, navigate to the corresponding page, press the add button, and select the desired item.
Within each page, right-clicking on a list item will present the option to either rename or remove it.
Individual items can be manually sorted by holding left-click over the selected item and dragging it.

*Note: Shortcuts can only be added and manipulated from their respective page. They cannot be modified from the home page.*

# Features
Settings:
- Show or hide the directory of items associated with selected shortcuts
- Enable or disable fixed window sizing
- Selection of available window sizes if Fixed Window Size is enabled
- Selection of available themes (Currently available themes: Default, Dark, Light)
